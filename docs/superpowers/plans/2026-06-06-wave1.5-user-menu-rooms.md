## Design tokens quick-reference

All inline styles must use these CSS tokens. Never use raw hex or rgba for color.

```
var(--bone)       #f4ecd8   primary text on dark bg
var(--bone-dim)   ~0.55α    secondary text
var(--bone-faint) ~0.35α    tertiary / placeholder
var(--warm)       #b07a4a   sealing-wax accent (<3% surface)
var(--rule)       ~0.07α    hairline borders
var(--ink)        #0e0e0c   base background
var(--mono)       JetBrains Mono
var(--serif)      Source Serif 4
```

Back-link pattern (canonical, used by LetterRoom/VoiceRoom):
```tsx
<Link
  to="/loom"
  style={{
    fontFamily: 'var(--mono)', fontSize: 10, letterSpacing: '0.12em',
    textTransform: 'uppercase', color: 'var(--bone-faint)', textDecoration: 'none',
  }}
>
  ← heirloom
</Link>
```
